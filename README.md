# AudioSwitcher v1.9.0.6 (SPluzh Fork)

A fork of the original [AudioSwitcher (v1)](https://github.com/FortyOne/AudioSwitcher) by FortyOne (xenolightning). 
This version brings modern improvements and features to the classic v1.x branch.

https://github.com/user-attachments/assets/94409893-cce7-4b8c-bc8c-7ad22bb6de98

## Key Features & Improvements
This fork includes several enhancements over the original project:

*   **Offline Mode**: Disabled all network activities, auto-update checks, donation links, and related UI elements.
*   **High DPI Support**: Improved UI scaling and clarity for modern high-resolution displays.
*   **Refined Volume Control**: Added a volume step hook feature allowing for precise **1% volume increments**.
*   **Classic Volume Mixer**: Option to open the classic Windows Volume Mixer from the tray context menu (also supports opening via **Middle Mouse Click**).
*   **WIN Key Modifier**: Proper support for using the **Windows Key** as a modifier for global hotkeys.
*   **Quick Switch Hotkey**: Added a global hotkey to cycle through audio output devices.
*   **Custom Device Icons**: Added the ability to change audio device icons to custom ones.
*   **Double Click Option**: Added an option to disable or configure the tray icon double-click behavior.

## Requirements
*   Windows 10 / 11
*   .NET Framework 4.8
